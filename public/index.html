FizzBuzz+

FizzBuzz is a classic programming challenge where the developer must write a program that prints each number from 1-100. For each multiple of 3, print "Fizz" instead of the number. For each multiple of 5, print "Buzz" instead of the number. For numbers that are a multiple of both 3 and 5 write "FizzBuzz".

In this new take on the classic challenge, the user can define the word, frequency, and color inputs. The display on the right will show the user's new output.
